## Usage

- Start the server by running: 

```sh
node server/server.js
```

- Start the app by running: 
```sh
expo start
```

## Requirements

- Node 10.15.3
- Postgresql 7.9.0
- Expo 32.0.0
- React 16.5.0
- Express 4.16.4
- Bcyrpt 3.0.4
